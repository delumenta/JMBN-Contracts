1.0 Purpose

This handbook defines the standard operational procedures for interacting with the JMBN Discord Automation Unit (DAU). All personnel must observe these guidelines to maintain clarity and discipline when retrieving mission data, confirming attendance, and communicating through approved channels.

2.0 System Overview

The DAU acts as the frontline interface for mission intelligence, enabling:

  • Mission Listing
  • Mission Detail Preview
  • RSVP Management
  • Account Linking
  • Profile Queries
  • Certification Display
  • Admin-controlled Mission Publishing

3.0 Account Linking Protocol

All personnel must link their Discord account before participating in missions.

3.1 Steps

  1. Open your JMBN Dashboard ‚Üí Account ‚Üí Link Discord.
  2. Generate a JMBN-XXXXXX link code.
